## Цель создания МРУ Юнга: 
Cпроектировать и изготовить модель мобильного роботизируемого устройства для профилактической чистки корпуса судна от органических обрастаний, без необходимости его постановки в док, которая будет производиться во время стоянки судна или на рейде (на малой скорости или дрейфе) силами экипажа, для оптимизации расходов на содержание судна.

## Задачи проекта: 
- Проанализировать основные методы борьбы с обрастанием  
- Проанализировать типы очистки корпуса корабля  
- Изучить используемые для очистки от обрастаний РУ. 
- Оценить эффективность МРУ.  
- Создать общую концепцию модели. 
- Создать чертеж устройства. 
- Выбрать материалы и комплектующие изделия. 
- Напечатать детали устройства на 3D-принтере. 
- Собрать устройство. 
- Запрограммировать устройство.
- Проверить работу устройства.  

## Общая концепция модели:
Технически МРУ представляет собой мобильную автоматизированную конструкцию, которая крепится на поверхности корпуса судна и перемещается по ней, удаляя грязь и обрастателей. Устройство предназначено для профилактической чистки, которая осуществляется за счет трения щёток о поверхность корпуса судна. 
Электрический двигатель создает и постоянно поддерживает вакуум, за счет которого устройство притягивается корпусу. Одновременно с притяжением происходит перемещение робота по поверхности и удаление загрязнений и прикрепившихся обрастателей. Робот передвигается с помощью треков (гусениц), изготовленных из резины, которые не повреждают поверхность корпуса.  
Питание робота осуществляется при постоянном подключении к электросети судна. Этот метод подачи энергии гарантирует стабильную и бесперебойную работу всех его систем, исключая необходимость в регулярной подзарядке или замене элементов питания. Резервным питанием робота служит аккумулятор. 
МРУ также снабжено стальным страховочным тросом.
